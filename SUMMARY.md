## 🎯 Ключевые компоненты

### Функции управления
```
openLesson()                    - открыть видео урок
initializeVideoTracking()       - инициализировать отслеживание
startTracking() / stopTracking() - начало/остановка счета
updateTrackingUI()              - обновить интерфейс
updateStatusIndicator()         - обновить индикатор
```

### Функции данных
```
saveWatchStats()                - сохранить в localStorage
getAllWatchStats()              - получить все данные
clearWatchStats()               - удалить данные
exportWatchStats()              - экспортировать JSON
formatTime()                    - преобразовать секунды в MM:SS
```

### Обработчики YouTube API
```
onYouTubeIframeAPIReady()       - YouTube API загружен
onPlayerReady()                 - плеер готов
onPlayerStateChange()           - изменение состояния
onPlayerError()                 - ошибка плеера
```

## 💾 Хранилище данных

### localStorage структура
```
Ключ:   "video_<название урока>"
Значение: {
  "totalSeconds": 245,
  "watchedAt": "2025-12-09T10:30:00.000Z",
  "lessonName": "название урока"
}
```

### Глобальные переменные
```
currentPlayer       - YouTube плеер
timerInterval       - ID интервала
totalSecondsWatched - время просмотра (сек)
videoDuration       - длительность видео (сек)
currentLesson       - название текущего урока
videoWatchStats     - кеш статистики
item_lessons        - база данных уроков
```

## 🔄 Жизненный цикл

```
1. ИНИЦИАЛИЗАЦИЯ
   - Загрузка course_content.js
   - YouTube API script добавляется

2. ОТКРЫТИЕ УРОКА
   - openLesson("название")
   - HTML генерируется и вставляется
   - initializeVideoTracking() стартует

3. ГОТОВНОСТЬ ПЛЕЕРА
   - YouTube API загружается
   - onPlayerReady() вызывается
   - videoDuration получается

4. ВОСПРОИЗВЕДЕНИЕ
   - onPlayerStateChange(PLAYING)
   - startTracking() запускает интервал
   - Каждую секунду: updateTrackingUI()

5. ПАУЗА
   - onPlayerStateChange(PAUSED)
   - stopTracking() останавливает интервал
   - Данные сохраняются в localStorage

6. ЗАКРЫТИЕ
   - beforeunload event
   - Финальное сохранение в localStorage
```

## ⚙️ Технические подробности

### YouTube API версия
- Используется: `https://www.youtube.com/iframe_api`
- Доступ: Глобальный объект `YT`
- События: onReady, onStateChange, onError
- Плеер: YT.Player класс

### localStorage
- Браузерное хранилище (5-50 MB в зависимости от браузера)
- Ключи: строки (используется JSON для данных)
- Доступность: все современные браузеры
- Очистка: при очистке истории браузера

### CSS особенности
- Flexbox и Grid для раскладки
- Linear gradient для прогресс-бара
- CSS анимация (pulse) для индикатора
- Адаптивная сетка (2 колонки → 1 на мобилях)

### JavaScript особенности
- ES6+ синтаксис (const, let, arrow functions)
- setInterval для счета времени
- JSON.stringify/parse для данных
- Promise/fetch для будущей интеграции

## 📞 Техническая поддержка

### Самые частые вопросы

**Q: Почему не работает отслеживание?**  
A: Проверьте что iframe имеет ID `my-youtube-player`

**Q: Где хранятся данные?**  
A: В `localStorage` браузера, ключи начинаются с `video_`

**Q: Как отправить на сервер?**  
A: Используйте `getAllWatchStats()` и `fetch(POST)`

**Q: Работает ли на мобилях?**  
A: Да, iOS Safari и Android Chrome поддерживают YouTube API

**Q: Можно ли скрыть блок отслеживания?**  
A: Да, добавьте CSS: `.video-tracking-container { display: none; }`
