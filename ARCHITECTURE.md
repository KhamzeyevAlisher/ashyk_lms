## 🔄 Поток данных

### 1. Инициализация

```
Загрузка страницы
    ↓
Загрузка course_content.js
    ↓
item_lessons загружается в память
videoWatchStats = {}
    ↓
YouTube API script добавляется в DOM
    ↓
onYouTubeIframeAPIReady() готово
```

### 2. Открытие урока

```
Пользователь кликает на урок
    ↓
openLesson("название")
    ↓
Получить данные из item_lessons
    ↓
Генерировать HTML (видео + отслеживание)
    ↓
Вставить в #tab-item-lesson
    ↓
initializeVideoTracking("название")
    ↓
Загрузить статистику из localStorage → videoWatchStats
    ↓
Найти #my-youtube-player iframe
    ↓
Создать YT.Player объект → currentPlayer
    ↓
Подключить обработчики событий
    ↓
Ожидание события onPlayerReady
```

### 3. Воспроизведение видео

```
Пользователь нажимает Play
    ↓
onPlayerStateChange(PLAYING)
    ↓
videoStartTime = new Date()
    ↓
startTracking()
    ↓
timerInterval = setInterval(function, 1000)
    ↓
Каждую секунду:
├─ totalSecondsWatched++
├─ updateTrackingUI()
│  ├─ Обновить #video-watch-time
│  ├─ Обновить #video-progress-percent
│  ├─ Обновить #video-duration-display
│  └─ Обновить #video-progress-bar ширину
└─ Каждые 5 сек: saveWatchStats()
   └─ Сохранить в localStorage
```

### 4. Остановка воспроизведения

```
Пользователь нажимает Pause/Video Ended/Buffering
    ↓
onPlayerStateChange(PAUSED/ENDED/BUFFERING)
    ↓
stopTracking()
    ↓
clearInterval(timerInterval)
    ↓
saveWatchStats() → localStorage
    ↓
updateStatusIndicator(false)
    ↓
Ожидание следующего Play
```

### 5. Закрытие страницы

```
Пользователь закрывает/уходит со страницы
    ↓
beforeunload event
    ↓
stopTracking()
    ↓
saveWatchStats() → localStorage (финальное сохранение)
    ↓
Страница выгружается
```

### 6. Возврат на страницу

```
Пользователь возвращается на страницу
    ↓
course_content.js загружается
    ↓
openLesson("название") вызывается
    ↓
initializeVideoTracking("название")
    ↓
localStorage.getItem("video_название")
    ↓
Если найдено:
├─ videoWatchStats[name] = JSON.parse(значение)
├─ totalSecondsWatched = сохраненное время
└─ UI обновляется с восстановленным временем
    ↓
Продолжить просмотр
```

## 🎯 Основные компоненты

### 1. **Управление плеером**

| Компонент | Назначение | Взаимодействует с |
|-----------|-----------|-----------------|
| `currentPlayer` | YT.Player объект | YouTube API, события |
| `videoDuration` | Длительность видео | onPlayerReady, updateTrackingUI |
| `onPlayerStateChange` | Обработчик событий | startTracking, stopTracking |

### 2. **Отслеживание времени**

| Компонент | Назначение | Взаимодействует с |
|-----------|-----------|-----------------|
| `timerInterval` | Интервал счета | startTracking, stopTracking |
| `totalSecondsWatched` | Счетчик времени | updateTrackingUI, saveWatchStats |
| `startTracking()` | Начало счета | onPlayerStateChange |
| `stopTracking()` | Остановка счета | onPlayerStateChange |

### 3. **Обновление UI**

| Компонент | Назначение | Обновляет |
|-----------|-----------|----------|
| `updateTrackingUI()` | Обновляет все элементы | DOM (6 элементов) |
| `formatTime()` | Форматирует время | Текстовые значения |
| `updateStatusIndicator()` | Обновляет точку статуса | Класс `.active` |

### 4. **Хранилище данных**

| Компонент | Назначение | Хранит |
|-----------|-----------|--------|
| `videoWatchStats` | Кеш в памяти | Статистика текущей сессии |
| `localStorage` | Постоянное хранилище | Статистика между сессиями |
| `item_lessons` | База уроков | Информация об уроках |

## 📊 Взаимодействие функций

### Цепочка вызовов при Play

```
YouTube API: onStateChange(PLAYING)
    ↓
startTracking()
    ├─ stopTracking() [очистить старый интервал]
    └─ setInterval(function, 1000)
        └─ Каждую секунду:
            ├─ totalSecondsWatched++
            ├─ updateTrackingUI(currentLesson)
            │  ├─ formatTime() [для всех временных значений]
            │  ├─ Обновить DOM элементы
            │  └─ Вычислить процент: (total / duration) * 100
            └─ Каждые 5 сек: saveWatchStats(currentLesson)
                └─ localStorage.setItem()
```

### Цепочка вызовов при Pause

```
YouTube API: onStateChange(PAUSED)
    ↓
stopTracking()
    ├─ clearInterval(timerInterval)
    └─ saveWatchStats(currentLesson)
        └─ localStorage.setItem()
```

## 🔐 Безопасность и надежность

### Проверки

```javascript
// 1. Проверка наличия урока
if (!lesson) console.error("Урок не найден");

// 2. Проверка наличия iframe
if (!iframe) console.error("iframe не найден");

// 3. Проверка YouTube API
if (typeof YT === 'undefined') {
  setTimeout(() => initializeVideoTracking(), 500);
}

// 4. Проверка localStorage
try {
  localStorage.setItem(...);
} catch(e) {
  console.error('localStorage недоступен');
}
```

### Восстановление при ошибках

- Если iframe не найден → UI все равно обновляется
- Если YouTube API не загружен → Повторная попытка через 500ms
- Если localStorage недоступен → Статистика остается в памяти
- Если видео невозможно загрузить → Выводится сообщение об ошибке

## 🚀 Масштабируемость

### Как добавить новое видео

1. Добавить в `item_lessons`:
```javascript
"Новый урок": {
  category: "категория",
  description: "описание",
  duration: "45 мин",
  date: "09.12.2025",
  iframe: '<iframe ...></iframe>',
  files: [...]
}
```

2. Вызвать:
```javascript
openLesson("Новый урок");
```

### Как масштабировать хранилище

Текущий подход использует localStorage каждого браузера. Для серверного хранилища:

```javascript
// Экспортировать статистику
const stats = exportWatchStats();

// Отправить на сервер
fetch('/api/save-watch-stats', {
  method: 'POST',
  body: JSON.stringify(stats)
});
```
