# 🎥 Система отслеживания времени просмотра видео

## 📌 Обзор

Полнофункциональная система отслеживания времени просмотра YouTube видео с визуальным интерфейсом и автоматическим сохранением данных в браузере.

**Статус:** ✅ Готово к использованию  
**Версия:** 1.0  
**Язык кода:** JavaScript (ES6+), HTML, CSS  
**Документация:** Полная (2700+ строк)

---

## 🎯 Что это делает?

Система отслеживает:
- ⏱️ Сколько времени пользователь смотрел видео
- 📈 Процент просмотра видео
- 🎬 Состояние видео (Play/Pause)
- 💾 Автоматически сохраняет данные

Пользователь видит:
- Текущее время просмотра
- Процент просмотра
- Визуальный прогресс-бар
- Индикатор активности видео

## 🚀 Быстрый старт (5 минут)

### 1. Откройте видео урок

```javascript
openLesson("Менеджменттің мәні мен қағидалары");
```

### 2. Пользователь смотрит видео

- Нажимает **Play** → счет начинается
- Нажимает **Pause** → счет останавливается
- Закрывает страницу → данные сохраняются

### 3. Получите статистику

```javascript
const stats = getAllWatchStats();
console.log(stats);
// Результат: объект со всеми видео и временем просмотра
```

## 💾 Где хранятся данные?

**localStorage** браузера

```
Ключ: "video_<название урока>"
Значение: JSON
{
  "totalSeconds": 245,
  "watchedAt": "2025-12-09T10:30:00.000Z",
  "lessonName": "название урока"
}
```

**Как посмотреть:**
1. Откройте DevTools (F12)
2. Перейдите: Application → Local Storage
3. Найдите ключи начинающиеся с `video_`

## 🌐 Интеграция с бэкендом

Если нужно отправить данные на сервер:

```javascript
// Получить статистику
const stats = getAllWatchStats();

// Отправить на сервер
fetch('/api/save-watch-stats', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(stats)
})
.then(res => res.json())
.then(data => console.log('Сохранено!'));
```

## 🐛 Часто задаваемые вопросы

**Q: Почему не работает отслеживание?**  
A: Проверьте что iframe имеет ID `my-youtube-player` и YouTube API загружен

**Q: Где хранятся данные?**  
A: В `localStorage` браузера, ключи начинаются с `video_`

**Q: Как отправить на сервер?**  
A: Используйте `getAllWatchStats()` и `fetch(POST)`

**Q: Работает ли на мобилях?**  
A: Да, iOS Safari и Android Chrome поддерживают YouTube API

**Q: Можно ли скрыть блок отслеживания?**  
A: Да, добавьте CSS: `.video-tracking-container { display: none; }`
