# 🧪 Примеры тестирования отслеживания видео

## 📌 Как тестировать функциональность

### 1. Открытие видео урока

```javascript
// В консоли браузера (F12)
openLesson("Менеджменттің мәні мен қағидалары");
```

**Ожидаемый результат:**
- Видео появляется на экране
- Показывается блок отслеживания времени
- В консоли: `YouTube плеер инициализирован`
- В консоли: `Длительность видео: 14:30`

---

### 2. Проверка отслеживания при проигрывании

```javascript
// 1. Откройте видео
openLesson("Менеджменттің мәні мен қағидалары");

// 2. Нажмите Play в видео плеере

// 3. В консоли должны появиться логи каждую секунду:
// "Пользователь смотрит: 1 сек"
// "Пользователь смотрит: 2 сек"
// и т.д.

// 4. Проверьте элементы UI:

document.getElementById('video-progress-percent').textContent; // "0%" или больше
document.getElementById('video-progress-bar').style.width; // "0%..." или больше
```

### 8. Очистка статистики

```javascript
// Очистить статистику конкретного видео
clearWatchStats("Менеджменттің мәні мен қағидалары");
console.log(getAllWatchStats()); // это видео не будет в списке

// Очистить всю статистику
clearWatchStats();
console.log(getAllWatchStats()); // {}
```

## 🧩 Чек-лист функциональности

### Базовое функционирование
- [ ] Видео загружается при вызове `openLesson()`
- [ ] Блок отслеживания времени видим на странице
- [ ] YouTube плеер инициализируется без ошибок

### Отслеживание времени
- [ ] При нажатии Play начинает считаться время
- [ ] Значение `#video-watch-time` увеличивается каждую секунду
- [ ] При нажатии Pause счет останавливается
- [ ] Счет возобновляется при повторном Play

### UI обновления
- [ ] `#video-watch-time` отображает правильный формат MM:SS
- [ ] `#video-progress-percent` показывает проценты (0-100%)
- [ ] `#video-duration-display` показывает общую длительность видео
- [ ] `#video-progress-bar` заполняется пропорционально прогрессу
- [ ] Прогресс-бар плавно растет (0% → 100%)

### Индикатор статуса
- [ ] Точка индикатора серая по умолчанию
- [ ] При Play точка становится зеленой
- [ ] При Play точка пульсирует (анимация)
- [ ] При Pause точка становится серой

### Сохранение данных
- [ ] localStorage содержит ключ `video_<название>`
- [ ] Данные сохраняются каждые 5 секунд
- [ ] При перезагрузке страницы статистика загружается
- [ ] При закрытии страницы данные сохраняются

### API функций
- [ ] `getAllWatchStats()` возвращает объект со всей статистикой
- [ ] `formatTime()` корректно преобразует секунды в MM:SS
- [ ] `clearWatchStats()` удаляет статистику из localStorage
- [ ] `exportWatchStats()` возвращает JSON строку

### Обработка ошибок
- [ ] При отсутствии iframe выводится ошибка в консоль
- [ ] При ошибке YouTube API выводится сообщение об ошибке
- [ ] Система не падает при повторном открытии видео
- [ ] Система не падает при отсутствии localStorage

## 🔍 Как отлаживать

### Включить подробное логирование

```javascript
// В консоли браузера добавьте перед openLesson():

// Перехватываем все изменения видео-переменных
const handler = {
  set(target, property, value) {
    console.log(`[VIDEO] ${property} изменился на:`, value);
    return true;
  }
};

// Обернуть глобальные переменные (если нужно)
// window.currentPlayer = new Proxy({}, handler);

// Или просто наблюдать переменные в консоли:
setInterval(() => {
  console.log({
    totalSecondsWatched,
    videoDuration,
    currentLesson,
    playerState: currentPlayer?.getPlayerState?.()
  });
}, 1000);
```

### Просмотр localStorage

```javascript
// В консоли браузера
console.table(localStorage);

// Или для видео-статистики:
Object.entries(localStorage).forEach(([key, value]) => {
  if (key.startsWith('video_')) {
    console.log(key, JSON.parse(value));
  }
});
```

### Проверка YouTube API

```javascript
// В консоли браузера
console.log(typeof YT); // "object"
console.log(YT.Player); // функция конструктора
console.log(YT.PlayerState); // объект с кодами состояний
```

## 🎯 Сценарии тестирования

### Сценарий 1: Полный просмотр видео

1. Открыть страницу с уроком
2. Вызвать `openLesson("Менеджменттің мәні мен қағидалары")`
3. Нажать Play в плеере
4. Подождать 10 секунд
5. Проверить: `#video-watch-time` показывает "0:10"
6. Нажать Pause
7. Проверить: счет остановился
8. Нажать Play еще раз
9. Подождать еще 10 секунд
10. Проверить: `#video-watch-time` показывает "0:20"
11. Закрыть страницу или вкладку
12. Вернуться на страницу и открыть то же видео
13. Проверить: статистика восстановлена из localStorage

### Сценарий 2: Проверка нескольких видео

1. Открыть первое видео
2. Просмотреть 30 секунд
3. Переключиться на второе видео
4. Просмотреть 45 секунд
5. В консоли: `getAllWatchStats()`
6. Проверить: оба видео в статистике с правильным временем

### Сценарий 3: Перезагрузка страницы

1. Открыть видео
2. Просмотреть видео
3. Нажать F5 для перезагрузки
4. Открыть то же видео еще раз
5. Проверить: статистика продолжает с сохраненной точки

### Сценарий 4: Очистка статистики

1. Получить статистику: `getAllWatchStats()`
2. Проверить: видео в списке
3. Вызвать: `clearWatchStats("название видео")`
4. Получить статистику еще раз
5. Проверить: видео удалено из списка

## 🐛 Частые проблемы и решения

### Проблема: Отслеживание не работает

**Возможные причины:**
1. YouTube API не загружен
2. iframe не имеет ID `my-youtube-player`
3. Неправильное название урока в `item_lessons`

**Решение:**
```javascript
// Проверить YouTube API
console.log(typeof YT !== 'undefined' ? 'API загружен' : 'API не загружен');

// Проверить iframe
console.log(document.getElementById('my-youtube-player') ? 'iframe найден' : 'iframe НЕ найден');

// Проверить урок
console.log(Object.keys(item_lessons));
```

### Проблема: localStorage не сохраняет

**Возможные причины:**
1. localStorage отключен в браузере
2. Приватный режим браузера
3. Браузер очистил данные

**Решение:**
```javascript
// Проверить localStorage
try {
  localStorage.setItem('test', 'test');
  localStorage.removeItem('test');
  console.log('localStorage работает');
} catch(e) {
  console.error('localStorage недоступен:', e);
}
```

### Проблема: Время скачет или неправильно

**Возможные причины:**
1. Несколько интервалов работают одновременно
2. Проблема с videoDuration (не загружена)

**Решение:**
```javascript
// Убедиться что интервал только один
console.log(timerInterval); // должен быть number или null

// Проверить длительность
console.log(videoDuration); // должна быть > 0

// Перезагрузить видео
openLesson("название");
```
