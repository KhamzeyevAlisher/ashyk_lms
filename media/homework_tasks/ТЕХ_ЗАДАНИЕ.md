## 5. Админ-панель
*   **5.1. Табы** (Главная, Пользователи, Структура университета, Настройка системы, Профиль).
*   **5.2. Вкладка "Главная"**
    *   5.2.1. Статистика (Общая).
    *   5.2.2. Статистика пользователей.
    *   5.2.3. Последняя активность.
*   **5.3. Пользователи**
    *   5.3.1. Кнопка "Импорт".
    *   5.3.2. Кнопка "Экспорт".
    *   5.3.3. Кнопка "Добавить пользователя".
    *   5.3.4. Фильтрация и поиск (Роли, статус, кафедра, Группа).
    *   5.3.5. Список пользователей (ФИО, Группа/Кафедра, Логин, Роль, Статус, Действие (CRUD)).
    *   5.3.6. Модальное окно для Импорта и экспорта.
    *   5.3.7. Модальное окно для добавление пользователя.
        *   ФИО.
        *   Роль.
        *   Email.
        *   Группа/Кафедра/факультет.
*   **5.4. Структура университета**
    *   5.4.1. Факультет.
        *   Список факультетов (Название, Декан, кол-во студентов, действие).
        *   Кнопка добавить факультет.
    *   5.4.2. Кафедра.
        *   Список кафедр (Название, Факультет, Заведующий, кол-во преподавателей, действие).
        *   Кнопка добавить факультет (ошибка в тексте, вероятно "кафедру").
    *   5.4.3. Группы.
        *   Список групп (Название, Факультет, Курс, кол-во студентов, куратор, действия).
        *   Кнопка добавить групп.
    *   5.4.4. Предметы.
        *   Список предметов (Название, Код, действия).
        *   Кнопка "Добавить".
    *   5.4.5. Назначения.
        *   Преподаватель — Предмет (список).
        *   (Преподаватель — предмет — действия).
        *   Кнопка добавить назначение на Предмет.
        *   Группа — Предметы список (Группа, Предмет, Преподаватель, часов, действия).
        *   Добавить назначение.
        *   Импорт учебного плана.
*   **5.5. Настройки системы**
    *   5.5.1. Сроки семестров.
    *   5.5.2. Назначение выходных дней.
    *   5.5.3. Логи активности.
*   **5.6. Профиль**
