# Документация модуля: Студенческие курсы (Student Courses)

Этот модуль отвечает за вывод сетки доступных курсов (дисциплин) в личном кабинете студента. Карточки генерируются динамически на основе массива данных.

## 1. Структура файлов

*   **`parties/student_dashboard/student_courses.html`** — HTML-шаблон компонента.
*   **`script.js`** — Логика рендеринга и данные.
*   **Main Container** — Родительский блок в основном файле с `id="tab-courses"`.

---

## 2. HTML Структура

Компонент должен быть обернут в родительский контейнер с уникальным ID, так как скрипт ищет конкретный селектор.

### Основные элементы:
*   **Контейнер обертка**: `<div id="tab-courses" class="courses-section ...">`.
*   **Заголовок**: `<h3>Барлық курстар</h3>` — Статический заголовок.
*   **Сетка курсов**: `<div class="courses-grid"></div>` — Пустой контейнер, куда JavaScript инъектирует карточки.

> **Важно**: Скрипт ищет элемент по селектору `#tab-courses .courses-grid`. Если изменить ID родителя или класс сетки, рендеринг не сработает.

---

## 3. Структура данных (JavaScript)

Данные хранятся в массиве `coursesData`.

### Пример объекта данных:

```javascript
{
    title: "Кәсіпкерлік",       // Название курса (используется как ID для клика)
    description: "Бизнес негіздері", // Краткое описание под заголовком
    image: "https://..."        // Ссылка на изображение обложки
}
```

---

## 4. Логика работы (JS функции)

### `renderCourses()`
Основная функция генерации разметки.

1.  **Поиск контейнера**: Ищет элемент `#tab-courses .courses-grid`. Если элемент не найден (например, на другой странице), функция безопасно завершает работу (`return`).
2.  **Очистка**: Очищает `innerHTML` контейнера во избежание дублирования при повторных вызовах.
3.  **Генерация карточек**:
    *   Проходит по массиву `coursesData`.
    *   Формирует HTML для каждой карточки (`.mini-course-card`).
    *   Добавляет атрибут `onclick="renderCourseByTitle('Title')"` для обработки клика.
4.  **Вставка**: Вставляет собранный HTML в DOM.

### Внешние зависимости
Код генерирует карточки с обработчиком клика:
```html
<div class="mini-course-card" onclick="renderCourseByTitle('Название курса')"> ... </div>
```
